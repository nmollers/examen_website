Realiseert software

In dit werkproces heb ik de website daadwerkelijk ontwikkeld op basis van het eerder gemaakte ontwerp. Hierbij heb ik verschillende functionaliteiten geïmplementeerd en de applicatie getest.

Wat heb ik gedaan?

Tijdens dit proces heb ik de 2 use cases ontwikkeld met HTML, CSS en JavaScript. Ik heb onder andere de 4 knoppen op de homepage en een contactformulier geïmplementeerd.

Mijn rol

Software developer – verantwoordelijk voor het realiseren van de 2 use cases van de website.

Gebruikte technieken

HTML CSS JavaScript Responsive design Form validatie

Tools

Visual Studio Code GitHub

Gerealiseerde functionaliteiten

  • Homepage met navigatie naar verschillende subpagina’s
  • Subpagina’s met informatie voor verschillende doelgroepen
  • Contactformulier waarmee gebruikers een bericht kunnen sturen
  • Responsieve layout zodat de website werkt op desktop en mobiel
  • Integratie van video’s op de website

Bewijsstukken

Use Case 1 – Contactformulier verzenden

Actor: Bezoeker

Preconditie

De gebruiker bevindt zich op de contactpagina.

Beschrijving

  1. De gebruiker opent de contactpagina
  2. De gebruiker vult zijn naam in
  3. De gebruiker vult zijn e-mailadres in
  4. De gebruiker schrijft een bericht
  5. De gebruiker klikt op verzenden
  6. Het systeem controleert de invoer
  7. Het systeem verstuurt het bericht
  8. Het systeem toont een bevestiging

Postconditie

Het bericht is succesvol verzonden.

Contactformulier implementatie

Voor de website heb ik een contactformulier ontwikkeld waarmee bezoekers eenvoudig contact kunnen opnemen. Het formulier bevat invoervelden voor naam, telefoonnummer, e-mail en een bericht. Wanneer een gebruiker het formulier verzendt, worden de gegevens via een PHP script (PHPMailer) verwerkt en verstuurd naar het e-mailadres van de opdrachtgever. Daarna krijgt de gebruiker een bevestiging dat het bericht succesvol is verzonden. 



<section id="contact-section" class="bg-[#b4955f] py-10">

  <div class="container mx-auto px-6 text-center">

    <h2 class="text-3xl font-bold text-white mb-6">
      Neem contact met ons op
    </h2>

    <p class="text-lg text-white mb-6">
      Heb je vragen of wil je meer informatie? Laat het ons weten!
    </p>

    <form action="./php/send-email.php" method="POST">

      <input type="text" name="name" required>

      <input type="tel" name="phone" required>

      <input type="email" name="email" required>

      <textarea name="message" required></textarea>

      <button type="submit">
        Verzenden
      </button>

    </form>

  </div>

</section>

Versiebeheer met Git en GitHub

Tijdens de ontwikkeling van de website heb ik gebruik gemaakt van Git voor versiebeheer. Git maakt het mogelijk om wijzigingen in de code bij te houden en eerdere versies van het project terug te halen. De repository van het project is opgeslagen op GitHub.

GitHub repository

De volledige broncode van het project staat opgeslagen in een GitHub repository. Hier worden alle bestanden van het project beheerd.
