Plant werkzaamheden en bewaakt voortgang

Planning maken, taken verdelen en voortgang bewaken.

Wat heb ik gedaan?

Voor dit werkproces heb ik een projectplanning gemaakt voor een webapplicatie. Ik heb user stories opgesteld en deze geprioriteerd volgens de MoSCoW methode. Daarnaast heb ik een sprint planning gemaakt en een Gantt chart gebruikt om de planning visueel weer te geven.

Mijn rol

Product Owner / Scrum Master – verantwoordelijk voor planning en voortgangsbewaking.

Gebruikte technieken

User Stories MoSCoW Prioritering Agile/Scrum Gantt Chart Sprint Planning

Tools

Jira Trello Microsoft Project Excel

Bewijsstukken

Sleep bestanden hierheen of klik om te uploaden

Reflectie

Wat ging goed?

De planning was realistisch en haalbaar. Door gebruik te maken van de MoSCoW methode kon ik duidelijk prioriteiten stellen.

Wat kon beter?

In het begin had ik te optimistische tijdsinschattingen. Ik had meer buffer moeten inbouwen.

Wat heb ik geleerd?

Ik heb geleerd dat een goede planning essentieel is voor het succes van een project.

Hoe pas ik dit toe in de toekomst?

In toekomstige projecten wil ik meer risico's en buffers inbouwen in mijn planning.

Resultaat / Eindproduct

Het resultaat is een complete projectplanning met user stories, sprint planning en een werkend Scrum board.
